# HBO Max Overlay Remover

---

## 🇬🇧 English

Removes the protection overlay layer from HBO Max player.

### 🚀 Features
- Automatically removes `protection_layer` element  
- Uses MutationObserver for dynamic updates  
- Lightweight and efficient  

### 📦 Installation
1. Install Tampermonkey  
   https://www.tampermonkey.net/

2. Install the script  
   Click the link below:
https://raw.githubusercontent.com/bulutcankaragoz/hbomax-overlay-remover/main/hbomax-overlay-remover.user.js

### ⚙️ How it works
The script observes DOM changes and removes:

[data-testid="protection_layer"]

This prevents the overlay from reappearing after re-renders.

### ⚠️ Disclaimer
This script only removes UI elements. It does not bypass DRM or any security systems.

## 🇹🇷 Türkçe

HBO Max oynatıcı üzerindeki koruma (overlay) katmanını kaldırır.

### 🚀 Özellikler
- `protection_layer` elementini otomatik kaldırır  
- Dinamik değişimler için MutationObserver kullanır  
- Hafif ve performanslıdır  

### 📦 Kurulum
1. Tampermonkey kur  
   https://www.tampermonkey.net/

2. Scripti yükle  
   Aşağıdaki linke tıkla:
https://raw.githubusercontent.com/bulutcankaragoz/hbomax-overlay-remover/main/hbomax-overlay-remover.user.js

### ⚙️ Nasıl çalışır
Script, DOM değişikliklerini izler ve şu elementi kaldırır:

[data-testid="protection_layer"]


Bu sayede yeniden render sonrası overlay tekrar oluşmaz.

### ⚠️ Uyarı
Bu script sadece arayüz elementlerini kaldırır. DRM veya güvenlik sistemlerini aşmaz.
